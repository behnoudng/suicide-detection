MentalBERT fine-tuned on 232k instances of suicidal and non-suicidal text. Creating an ensembled prediction system with XGBoost that achieves up to 99% recall on predicting suicidal tendencies in text.

## Dataset
This project uses [Suicide and Depression Detection Dataset](https://www.kaggle.com/datasets/nikhileswarkomati/suicide-watch) licensed under CC BY-SA 4.0.
